# Swapify

A skill exchange platform where users can teach and learn from each other.

## Features

- User authentication
- Skill matching system
- Real-time feedback
- Contact form for developer support
- Developer dashboard for managing feedback

## Setup

### Backend
```bash
cd swapify-backend
npm install
npm start
```

### Frontend
```bash
cd swapify-frontend
npm install
npm start
```

## Developer Access

## Technologies Used
- Frontend: React.js
- Backend: Node.js, Express
- Database: MongoDB
